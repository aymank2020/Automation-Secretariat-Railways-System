## 💻 كيفية تشغيل النظام على جهازك

### الطريقة السريعة (Docker) 🐳

```bash
cd Automation-Secretariat-Railways-System
./start.sh
```

أو يدوياً:
```bash
docker-compose up -d
docker-compose exec backend python seed_db.py
```

**افتح:**
- 🌐 http://localhost:5173 (الفرونت إند)
- 📖 http://localhost:8000/docs (API Docs)

**تسجيل الدخول:**
- بيانات المسؤول الأولي تُحدَّد من ملف `.env` عبر المتغيرين `INITIAL_ADMIN_USERNAME` و `INITIAL_ADMIN_PASSWORD`.
- لا توجد كلمات مرور افتراضية مدمجة في الكود.

### الطريقة اليدوية (بدون Docker)

**الباك إند:**
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python seed_db.py
uvicorn app.main:app --reload
```

**الفرونت إند:**
```bash
cd frontend
npm install
npm run dev
```
