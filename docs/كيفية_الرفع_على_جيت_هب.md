# 🚀 كيفية رفع المشروع على GitHub

المشروع جاهز بالكامل ومحفوظ محلياً! 🎉

## الخطوات النهائية للرفع:

### الطريقة 1: باستخدام Personal Access Token (موصى به)

1. **إنشاء GitHub Personal Access Token**
   - افتح: https://github.com/settings/tokens
   - اضغط "Generate new token" → "Generate new token (classic)"
   - حدد الصلاحيات: `repo` (كاملة)
   - اضغط "Generate token"
   - انسخ التوكن (سنحتاجه)

2. **رفع المشروع**
```bash
cd Automation-Secretariat-Railways-System

# أضف الـ remote (لو مش مضاف من قبل)
git remote add origin https://YOUR_TOKEN@github.com/aymanbayoumi080-oss/Automation-Secretariat-Railways-System.git

# أو لو مضاف بالفعل
git remote set-url origin https://YOUR_TOKEN@github.com/aymanbayoumi080-oss/Automation-Secretariat-Railways-System.git

# ارفع المشروع
git push -u origin main
```

> **مهم**: استبدل `YOUR_TOKEN` بالتوكن اللي نسخته

### الطريقة 2: باستخدام SSH Key (أكثر أماناً)

1. **تكوين Git ببياناتك**
```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

2. **رفع باستخدام SSH**
```bash
cd Automation-Secretariat-Railways-System
git remote set-url origin git@github.com:aymanbayoumi080-oss/Automation-Secretariat-Railways-System.git
git push -u origin main
```

### بعد الرفع بنجاح ✅

ستجد المشروع هنا:
👉 **https://github.com/aymanbayoumi080-oss/Automation-Secretariat-Railways-System**

## 📦 كيفية التشغيل على جهازك

### باستخدام Docker (الأسهل):

1. **استنساخ المشروع**
```bash
git clone https://github.com/aymanbayoumi080-oss/Automation-Secretariat-Railways-System.git
cd Automation-Secretariat-Railways-System
```

2. **إعداد البيئة**
```bash
cd backend
cp .env.example .env
# عدّل SECRET_KEY في ملف .env
```

3. **تشغيل المشروع**
```bash
cd ..
docker-compose up -d
```

4. **إنشاء قاعدة البيانات**
```bash
docker-compose exec backend python seed_db.py
```

5. **فتح النظام**
   - افتح المتصفح على: `http://localhost:5173`
   - تسجيل الدخول: بيانات المسؤول الأولي تُحدَّد من ملف `.env` عبر `INITIAL_ADMIN_USERNAME` و `INITIAL_ADMIN_PASSWORD`.

### بدون Docker:

**تشغيل الباك إند:**
```bash
cd backend
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
# عدّل DATABASE_URL و SECRET_KEY
python seed_db.py
uvicorn app.main:app --reload
```

**تشغيل الفرونت إند:**
```bash
cd frontend
npm install
npm run dev
```

افتح: `http://localhost:5173`
